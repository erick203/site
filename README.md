# 💻 Portfólio Pessoal - Erick Monteiro

Este repositório contém o código-fonte do meu portfólio profissional. O projeto foi desenvolvido com foco em performance, acessibilidade e design responsivo, utilizando as melhores práticas de desenvolvimento Web.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica para melhor SEO e acessibilidade.
- **CSS3:** Estilização moderna com uso de variáveis dinâmicas (`:root`), Flexbox e Grid Layout.
- **Font Awesome:** Integração de ícones para uma interface intuitiva.
- **Google Fonts:** Tipografia otimizada para leitura.

## 🏗️ Estrutura do Código
O desenvolvimento foi guiado por princípios de organização que facilitam a manutenção:

- **Arquitetura CSS:** Uso de variáveis globais para cores e fontes, permitindo alterações rápidas em todo o projeto.
- **Design Responsivo:** Implementação de `@media queries` para garantir que o site funcione perfeitamente em desktops, tablets e smartphones.
- **Interatividade:** Efeitos de `hover` e transições suaves (`transition`) para melhorar a experiência do usuário (UX).
- **Semântica HTML:** Uso de tags como `<nav>`, `<section>`, `<article>` e `<footer>` para uma estrutura limpa e compreensível por motores de busca.

## 📂 Organização de Arquivos
- `index.html`: Estrutura principal do site.
- `style.css`: Toda a lógica de estilo e responsividade.
- `/fotos`: Diretório destinado a imagens e logos.
- `/documentos`: Local reservado para os certificados em PDF.
